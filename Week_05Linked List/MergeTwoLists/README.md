# Merge Two Sorted Lists

## Idea

* عندك **لستين مرتبّتين**.
* المطلوب: ندمجهم في **لست واحدة مرتبة**.
* **Pattern:** Dummy Node لتسهيل التعامل مع الـ head.
* نستخدم pointer واحد (`current`) لبناء اللست الجديدة.
* نقارن كل خطوة، نضيف الأصغر، ونكمل.
* بعد ما واحدة تخلص، نضيف باقي الليست التانية كلها.

## Example

```
l1: 1 → 3 → 5
l2: 2 → 4 → 6
Result: 1 → 2 → 3 → 4 → 5 → 6
```

## Complexity

* **Time:** O(n + m)
* **Space:** O(1)
