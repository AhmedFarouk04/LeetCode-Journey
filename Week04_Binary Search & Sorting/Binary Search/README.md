

## 📌 Binary Search – README

### 🧩 Problem

**Binary Search**
Given a **sorted array** of integers and a target value, return the index of the target if it exists, otherwise return `-1`.

---

### 🧠 Key Idea

* Sorted Array
*  not used Linear Search o(n)
---

### 🔍 Pattern Used

* **Binary Search**
* Divide & Conquer
* Search in sorted array

---

### ⏱️ Complexity

* **Time:** `O(log n)`
* **Space:** `O(1)`
